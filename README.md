# AR_spawning
My first experience with Augmented Reality Development on Unity. The application tracks images as markers to spawn respective Models.

### Instructions
1) Download & Install Application on Android Phone
2) Print out the Markers
3) Place Markers on flat surfaces and direct camera to them to see the Models.
